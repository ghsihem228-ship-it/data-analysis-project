# Student Performance Analysis

## Overview
This project analyzes student academic performance using Python.  
It includes data loading, statistical analysis, and visualization of average scores.

## Tools Used
- Python
- Pandas
- Matplotlib
- CSV Data

## Files
- `data.csv`: Student performance dataset
- `analysis.py`: Basic analysis script
- `analysis_advanced.py`: Advanced analysis with visualization

## Objectives
- Calculate average scores by subject
- Identify top and lowest performers
- Visualize performance trends

## Key Insights
- Mathematics, Physics, and English scores were compared
- Average scores were calculated for each subject
- A bar chart was generated to visualize subject performance
